# Template do Briefing de Campanha

> **O que é.** O **Briefing de Campanha** é o artefato intermediário que consolida todas as decisões estratégicas de uma campanha num único documento. É **produzido pelo Agente Estrategista** a partir do Arquivo Base de Curso (e de pesquisa de audiência, quando houver) e é o **insumo único do Agente Copywriter** — que opera só sobre ele, sem voltar ao Arquivo Base.
>
> **Por que existe.** Separa o *pensar* (estratégia, feita uma vez por campanha) do *escrever* (copy, feita muitas vezes). Permite **múltiplas versões** — briefings diferentes para o mesmo produto (outra Big Idea, outro ICP, outro lançamento) coexistem como artefatos distintos.
>
> **O que NÃO entra aqui.** O **Arquivo de Voz** (DNA de escrita do expert/marca) é **separado** e pré-existente: a voz é da *marca* (estável, reusável entre campanhas) e o briefing é da *campanha* (muda a cada produto). O copywriter lê os dois: **Briefing + Arquivo de Voz + módulo do formato**.
>
> **Como o Estrategista preenche.** Cada seção abaixo é alimentada por um módulo da `base/`. O estrategista roda o módulo, conduz o usuário às escolhas, e registra aqui a decisão (não todas as alternativas — só o que foi escolhido, com as alternativas relevantes em standby quando útil). Campos inferidos devem ser marcados `[inferido — confirmar]`.
