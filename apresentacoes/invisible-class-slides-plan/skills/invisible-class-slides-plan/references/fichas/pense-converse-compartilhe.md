# FICHA — Pense-converse-compartilhe

**Família:** G — Processamento ativo · **ID:** `pense-converse-compartilhe`

**Função didática.** Estruturar uma atividade em três tempos (think-pair-share): o aluno **pensa sozinho**, **discute em par** e a sala **compartilha**. O trabalho na cabeça do aluno: formular a própria resposta antes de ouvir a dos outros, testá-la num par e refiná-la no coletivo — três passadas sobre a mesma ideia, cada uma fixando mais.

**Posição no arco.** Meio do Desenvolver, depois de um conceito que merece elaboração. É o respiro ativo mais profundo do arco — exige mais tempo que enquete ou predição.

**Quando usar.** Quando a ideia ganha em ser discutida, não só recebida; quando há tempo de aula para os três tempos; quando você quer que vozes além das mais rápidas apareçam (o "pense sozinho" protege quem precisa de tempo).

**Quando NÃO usar.** Em aula curta ou corrida (os três tempos não cabem). Para checagem rápida de acerto → [verificacao-entendimento](verificacao-entendimento.md); para sondagem instantânea → [enquete](enquete.md). Quando a pergunta não rende discussão (resposta única e óbvia).

**Anatomia (slots).**
- *A tarefa* (obrigatório) — uma pergunta/problema que vale discutir, clara o bastante para o trabalho solo.
- *Os três tempos* (obrigatório) — Pense / Converse / Compartilhe rotulados, cada um com seu papel.
- *O tempo de cada etapa* (opcional, recomendado) — minutos por fase, visíveis, para a sala se organizar.
- *O foco do compartilhar* (opcional) — o que a sala traz de volta (uma conclusão, uma divergência).

**Regras de carga.** Uma tarefa por slide. Os três rótulos visíveis sem poluir. Enunciado curto. Sem despejar a "resposta certa" na tela — ela emerge do compartilhar.

**Build.** Pode revelar os tempos em sequência (foca a sala na fase atual) ou exibir os três de uma vez como roteiro. O intervalo das três fases é a atividade — o slide só a emoldura e marca o tempo.

**Decisão visual (roteamento).** **Modo 0** sempre — três blocos rotulados, tipografia grande, talvez ícone funcional por fase (não decorativo). Não roteia para gerador → ver [../producao-visual.md](../producao-visual.md).

**Conexão posicional.** *Pede antes:* o conceito que será elaborado na atividade. *Aponta para:* a síntese do que a sala trouxe — o compartilhar alimenta o slide seguinte (um [takeaway](takeaway.md) ou [sintese](sintese.md)).

**Erro comum.** Pular o "pense sozinho" e ir direto pra conversa (os rápidos dominam, o resto copia). Não marcar tempo (a atividade se arrasta ou some). Pergunta sem miolo para discutir. Não fazer nada com o que a sala compartilha.

**Por dial de fidelidade.** *Transcrição:* se o professor conduziu uma dinâmica de pares/grupos, reconstrua a tarefa e a estrutura dela. Se não, você pode propor o formato sobre o ponto, marcado como adição. *Esboço:* você desenha a tarefa que melhor rende discussão no bloco e define os tempos.
