# FICHA — Matriz 2×2 / quadrante

**Família:** C — Estrutura e relações · **ID:** `matriz-2x2`

**Função didática.** Posicionar itens em **dois eixos cruzados**, gerando quatro quadrantes, para o aluno ler relações por posição. O trabalho na cabeça do aluno: ver que duas dimensões combinadas criam categorias qualitativamente distintas — e onde cada item cai.

**Posição no arco.** Meio (Desenvolver). Depois de os dois eixos (as duas variáveis) terem sido nomeados e entendidos isoladamente.

**Quando usar.** Há exatamente **duas** variáveis relevantes e cruzá-las revela algo (quadrantes com nome/significado); quando a posição relativa dos itens ensina mais que uma lista.

**Quando NÃO usar.** Uma só dimensão entre dois extremos → [espectro-continuum](espectro-continuum.md). Pesar prós e contras de uma opção → [trade-off-pros-contras](trade-off-pros-contras.md). Mais de dois eixos → não cabe num 2×2; simplifique ou outro tipo. Eixos que não se cruzam de forma significativa → tabela.

**Anatomia (slots).** *Título-asserção* (obrig., afirma o que o cruzamento revela, não "Matriz de X por Y"). *Dois eixos rotulados* (obrig., com os polos nomeados em cada ponta). *Quatro quadrantes* (obrig., idealmente com rótulo/nome curto cada). *Itens plotados* (opc., pontos posicionados nos quadrantes). *Quadrante em foco* (opc., destaque do que importa).

**Regras de carga.** Dois eixos, quatro quadrantes — nunca mais; máx. ~2-3 itens plotados por quadrante; rótulo de quadrante = nome curto, não definição; polos dos eixos claros e opostos.

**Build.** Recomendado: desenha os eixos vazios → nomeia os quadrantes um a um → plota os itens por último. Deixa o aluno construir o mapa antes de povoá-lo.

**Decisão visual (roteamento).** **Modo 0 por padrão** — eixos, grade e rótulos são estrutura precisa, resolve em HTML/CSS/SVG. *Nunca* manda para gerador de imagem: erra rótulos de eixo e desalinha a grade. Ver [../producao-visual.md](../producao-visual.md).

**Conexão posicional.** *Pede antes:* as duas dimensões já apresentadas separadamente. *Aponta para:* uma decisão por quadrante, uma estratégia, ou uma síntese.

**Erro comum.** Eixos não ortogonais (medem a mesma coisa) → quadrantes sem sentido. Quadrantes sem nome (o aluno não sabe o que cada um significa). Itens demais virando nuvem ilegível. Título-tópico.

**Por dial de fidelidade.** *Transcrição:* use os eixos e o posicionamento que o professor traçou; não reposicione itens nem renomeie quadrantes. *Esboço:* pode propor os eixos e os nomes dos quadrantes, marcados como gerados para aprovação.
