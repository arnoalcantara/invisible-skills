# FICHA — Recursos / referências

**Família:** H — Reforço e fechamento · **ID:** `recursos-referencias`

**Função didática.** Dizer para onde ir depois: leituras, fontes, links, próximos passos. O trabalho na cabeça do aluno: levar uma trilha para aprofundar fora da aula. Diferente de quase todos os outros tipos, este é um **slide de handout** — pensado para ser consultado depois, não absorvido ao vivo. Por isso mais texto é tolerável aqui: é referência, não projeção densa que disputa com a fala.

**Posição no arco.** Fechamento, perto do fim. Último ou penúltimo slide. Também pode encerrar um bloco grande, indicando aprofundamento daquele tema específico.

**Quando usar.** Ao fechar a aula (ou um bloco) com fontes, bibliografia, links e próximos passos que o aluno deve levar consigo.

**Quando NÃO usar.** No meio do conteúdo (interrompe o fluxo — referência é destino, não trânsito). Para a ideia que o aluno leva na cabeça → [takeaway](takeaway.md). Para reunir os fios do conteúdo → [sintese](sintese.md).

**Anatomia (slots).**
- *Lista de recursos* (obrigatório) — fontes, leituras, links, ferramentas. Organizada (por tipo ou prioridade).
- *Curadoria* (recomendado) — uma marca do essencial vs. opcional ("comece por"), para não afogar o aluno.
- *Próximos passos* (opcional) — o que fazer a seguir, se houver trilha.
- *Forma de acesso* (opcional) — QR code, link curto, onde baixar o material.

**Regras de carga.** A exceção controlada: mais itens são aceitáveis (é handout), mas ainda assim curados — marque o essencial, não despeje tudo. Agrupe por tipo (lei 10). Legibilidade mantida: mesmo como referência, projetado ao vivo precisa ser lido (lei 11). Se a lista for enorme, aponte para um documento externo em vez de enchê-la na tela.

**Build.** Estático. É consulta, não revelação — mostre completo.

**Decisão visual (roteamento).** **Modo 0** — lista tipográfica organizada do design system, eventual QR code (gerado em código/SVG). Sem imagem decorativa. Capa de livro (Modo 1, stock) só se ajudar a identificar a fonte. Ver [../producao-visual.md](../producao-visual.md).

**Conexão posicional.** *Pede antes:* todo o conteúdo da aula — as referências aprofundam o que foi visto. *Aponta para:* fora da aula — o estudo autônomo do aluno. É a ponte para depois.

**Erro comum.** Despejar uma bibliografia inteira sem curadoria (afoga em vez de guiar). Links que ninguém vai copiar de uma projeção (use QR ou link curto). Esconder no meio da aula. Tratar como slide de projeção densa e espremer fonte ilegível.

**Por dial de fidelidade.** *Transcrição:* liste apenas as fontes que o professor de fato citou ou indicou; não invente referências para "completar" — referência fabricada é erro grave. *Esboço:* você pode sugerir recursos pertinentes, marcando-os como sugestão sua a verificar, separados dos que o professor indicou.
